# OK-OK
This project aims to address the inefficiency of having to look up multiple platforms in finding affordable ukay-ukay items. This project aims to create a web app that centralizes ukay-ukay transactions, enabling customers and sellers sharing the same passion regarding the culture to have a platform to engage.
